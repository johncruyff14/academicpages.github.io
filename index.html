About FusionThor

FusionThor, an innovative dataset based on AI2Thor, is designed to rigorously evaluate a range of capabilities in downstream AI models. It tests text comprehension, including understanding scene hints and object labels, favoring models with strong OCR capabilities. It assesses Affordance understanding, including object relationships and common sense reasoning, benefiting models with robust Semantic Map capabilities, LLM query abilities, or good global mapping capabilities. It examines spatial relationship recognition, including scene layout understanding and object finding in static occlusion scenarios, favoring models based on LayoutGPT or 3D Scene Graph. It evaluates dynamic occlusion coping and obstacle avoidance abilities, rewarding models with strong global mapping or dynamic path planning capabilities. Lastly, it tests open-world object detection, particularly the identification of unusual objects, favoring large models capable of recognizing new objects rather than overfitting on trained ones.
